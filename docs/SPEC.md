# Functional Specifications

## 🌍 Domain Context

AOTypes is designed to provide a standardized, type-safe foundation for multiplatform applications. Its primary goal is to eliminate data duplication and typing errors across common quantitative and financial domains.

### Public Identity
The library is distributed globally via Maven Central under the following coordinates:
- **Group ID**: `io.github.aughtone`
- **Artifact ID**: `types`

This ensures that any KMP project can integrate these core types with a consistent dependency path.
